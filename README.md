# Мой учебный проект

## Description

### Задачи проекта:
* Вывод информации о студенте
* Демонстрация работы с Git
* Оформление документации в Markdown



![Логотип Git](https://git-scm.com/images/logo@2x.png)


Это учебный проект, который выводит информацию о студенте и его группе.
Программа написана на языке C и демонстрирует базовые навыки работы с Git и Markdown.
Проект создан в рамках контрольной работы.

## Installation
1. Склонируйте репозиторий: `git clone https://github.com/Nawruzbek/ibragimov-navruzbek.-kr-1`
2. Перейдите в директорию проекта: `cd ibragimov-navruzbek.-kr-1`
3. Установите компилятор GCC (если не установлен): `brew install gcc` (для macOS)

## Usage

**Важно:** Перед компиляцией убедитесь, что установлен компилятор *GCC* или *Clang*.


Приложение выводит информацию о студенте и группе.

Для компиляции и запуска выполните:
```bash
gcc src/main.c -o program
./program
```
